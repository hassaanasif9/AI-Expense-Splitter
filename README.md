### 🔹 App Overview  
This app allows users to track expenses

### 🛠 Installation Guide  
1. Clone this repository:  
git clone <your_repo_url>

2. Install dependencies:  
pip install -r requirements.txt

3. Run the app:  
streamlit run app.py
